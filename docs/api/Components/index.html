Components

Very simple components fetcher module to get react components we make

Types​

Component​

</>
type Component = {[string]:any}

React component

Theme​

</>
type Theme = {[string]:any}

Theming table, see default theme in src/ReplicatedStorage/Themes/Default.lua for a template to follow.

Properties​

</>

React context for current theme.

Fires when the theme gets updated

Theme​

Theme component controller

Functions​

GetComponent​

</>
Components:GetComponent(Name: string) → Component

Gets a themed version of a roblox base component such as a textbutton, frame, imagebutton.

GetController​

</>
Components:GetController(Name: string) → Component

Returns a controller object. Anything that the player interacts with such as sliders, dropdowns, radios.

GetBase​

</>
Components:GetBase(Name: string) → Component

Gets a base component. This includes things such as Window, Document

GetPadding​

</>
Components:GetPadding(Name: string) → Component

Gets a padding component. This includes things such as Box, Text

GetMain​

</>
Components:GetMain(Name: string) → Component

Gets a main component. This is a convienent way of getting your UI that you built.

SetTheme​

</>
Components:SetTheme(theme: Theme) → nil

Sets the UI theme. See default theme in src/ReplicatedStorage/Themes/Default.lua for a template to follow.
